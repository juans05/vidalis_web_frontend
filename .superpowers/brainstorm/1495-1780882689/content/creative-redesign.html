📐 Design System Applied:

  • Typography: Fredoka (headings - playful) + Nunito (body - friendly)
  • Primary Colors: Rose #E11D48 + Pink #FB7185 + Blue #2563EB
  • Style: Vibrant & Block-based con formas orgánicas
  • Effects: Smooth transitions 0.2-0.3s, hover scale, shadows enhancing
  • Animation: Subtle: hover translateY, focus states, smooth color changes
  • Spacing: 8dp system - 12px / 16px / 24px / 40px gaps
  • Accessibility: Focus states visibles, contrast 4.5:1, keyboard navigation
